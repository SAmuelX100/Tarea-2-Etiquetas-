Etiquetas HTML y sus Definiciones y Uso en HTML5

# Etiqueta Descripción Ejemplo de uso
89 <q> Indica que el texto adjunto es una cita corta en línea. Ver Ejemplo
90 <rp> Define el contenido que se mostrará antes y después del contenido ruby. Ver Ejemplo
91 <rt> La <rt>etiqueta define una explicación o pronunciación de caracteres (para tipografía de Asia Oriental) en una anotación rubí. Ver Ejemplo
92 <s> Muestra el texto tachado con una linea horizontal. Ver Ejemplo
93 <samp> Se utiliza para incluir texto en línea que representa una muestra (o cita) de la salida de un programa de ordenador. Ver Ejemplo
94 <script> La <script> etiqueta se utiliza para incrustar un script del lado del cliente (JavaScript). Ver Ejemplo
95 <search> La <search> etiqueta se utiliza para especificar que aquí viene un conjunto de elementos relacionados con la búsqueda. Ver Ejemplo
96 <select> Representa un control que muestra un menú de opciones. Las opciones contenidas en el menú son representadas por elementos <option>, los cuales pueden ser agrupados por elementos <optgroup>. Ver Ejemplo
97 <source> Especifica recursos de medios múltiples para los elementos <picture>, <audio>, o <video>. Ver Ejemplo
98 <span> Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea. Ver Ejemplo
99 <style> Es el elemento encargado de indicar la información de estilo. Ver Ejemplo
100 <sub> Define un fragmento de texto que se debe mostrar, por razones tipográficas, más bajo, y generalmente más pequeño, que el tramo principal del texto, es decir, en subíndice. Ver Ejemplo
101 <template> Mecanismo para mantener el contenido HTML del lado del cliente que no se renderiza cuando se carga una página, pero que posteriormente puede ser instanciado durante el tiempo de ejecución empleando JavaScript. Ver Ejemplo
102 <textarea> Representa un control para la edición mutilínea de texto sin formato. Ver Ejemplo
