Etiquetas HTML y sus Definiciones y Uso en HTML5

# Etiqueta Descripción Ejemplo de uso
23 <mark> Representa un texto marcado o resaltado como referencia o anotación, debido a su relevancia o importancia en un contexto particular. Ver Ejemplo
24 <em> Apropiado para marcar con énfasis las partes importantes de un texto. Ver Ejemplo
25 <strong> Apropiado para marcar con especial énfasis las partes más importantes de un texto. Ver Ejemplo
26 <b> Indica que el texto debe ser representado con una variable bold, o negrita,de la tipografía que se esté usando. Ver Ejemplo
27 <small> Hace el tamaño del texto una talla más pequeña (por ejemplo, de largo a mediano, o de pequeño a extra pequeño) que el tamaño mínimo de fuente del navegador. Ver Ejemplo
28 <cite> Marca una referencia a una fuente, o el autor de un texto citado. Ver Ejemplo
29 <address> Este elemento encierra información de contacto para un elemento <article> o el documento completo. Es recomendable que sea insertado dentro de un elemento <footer> Ver Ejemplo
30 <time> Representa un periodo específico en el tiempo. Puede incluir el atributo "datetime" para convertir las fechas en un formato interno legible por un ordenador, permitiendo mejores resultados en los motores de búsqueda o características personalizadas como recordatorios. Ver Ejemplo
31 <br> Para indicarle al navegador que queremos que continúe en la próxima línea debemos hacerlo con el elemento HTML. Ver Ejemplo
32 <ol> Permite definir listas o viñetas ordenadas ("Ordered List"), bien con numeración o alfabéticamente. Ver Ejemplo
33 <ul> Una lista no ordenada como su nombre lo indica no utiliza un número delante de cada items sino un pequeño símbolo gráfico. Ver Ejemplo
34 <dl> Como su nombre lo indica se utiliza para asociar un término y la definición del mismo. El navegador se encarga de destacar y separa el término y su definición. Ver Ejemplo
35 <form> Representa una sección de un documento que contiene controles interactivos que permiten a un usuario enviar información a un servidor web. Ver Ejemplo
36 <pre> Nos permite mostrar toda la información que está contenida entre las marcas de comienzo y fin tal cual como lo hemos escrito respetando inclusive los espacios en blanco y salto de líneas. Ver Ejemplo
37 <abbr> Representa una abreviación o acrónimo; el atributo opcional "title" puede ampliar o describir la abreviatura. Ver Ejemplo
38 <acronym> Es el encargado para marcar estas formas abreviadas (modem, AJAX...). Además, gracias al atributo "title" podemos indicar la versión extendida del término. Ver Ejemplo
39 <video> Se utiliza para incrustar vídeos en un documento HTML o XHTML. Ver Ejemplo
40 <area> Define un área dentro de un mapa de imagen que tiene áreas cliqueables predefinidas. Ver Ejemplo
41 <wbr> Representa una posición dentro del texto donde el explorador puede opcionalmente saltar una línea , aunque sus reglas de salto de línea de otra manera no crearían un salto en esa posición. Ver Ejemplo
42 <audio> Se usa para insertar contenido de audio en un documento HTML o XHTML. Ver Ejemplo
43 <base> Especifica la dirección URL base que se utilizará para todas las direcciones URL relativas contenidas dentro de un documento. Ver Ejemplo
44 <bdi> Aisla un trozo de texto para que pueda ser formateado con una dirección diferente al texto que hay fuera de él. Ver Ejemplo
