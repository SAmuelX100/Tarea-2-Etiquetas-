Etiquetas HTML y sus Definiciones y Uso en HTML5

# Etiqueta Descripción Ejemplo de uso
1 <!DOCTYPE html> Define el documento como HTML5. Ver Ejemplo
2 <html> Es el elemento raíz de un documento HTML. Ver Ejemplo
3 <head> Contiene la información del documento, como el título, los estilos CSS y las referencias a scripts JavaScript. Ver Ejemplo
4 <title> Agrega un título al documento. Se muestra en la pestaña del navegador o en las aplicaciones que lo utilizan. Ver Ejemplo
5 <body> Contiene el contenido visible de la página web, como texto, imágenes, vídeos, etc. Ver Ejemplo
6 <h1> a <h6> Encabezados de diferentes niveles de importancia, donde <h1> es el más importante y <h6> el menos. Ver Ejemplo
7 <p> Se utiliza para agregar párrafos de texto. Ver Ejemplo
8 <a> Se utiliza para crear enlaces a otras páginas web o recursos. Ver Ejemplo
9 <img> La Etiqueta img es una etiqueta de imagen que se utiliza para insertar una imagen en un documento. Ver Ejemplo
10 <meta> Hay varios tipos de etiqueta <meta> que pueden ser incluidas para declarar información general sobre el documento, pero esta información no es mostrada en la ventana del navegador, es solo importante para motores de búsqueda y dispositivos que necesitan hacer una vista previa del documento u obtener un sumario de la información que contiene. Ver Ejemplo
11 <link> Este elemento es usado para incorporar estilos, códigos Javascript, imágenes o iconos desde archivos externos. Uno de los usos más comunes para es la incorporación de archivos con estilos CSS. Ver Ejemplo
12 <table> Se utiliza para crear tablas en una página web. Las tablas son estructuras de datos que organizan la información en filas y columnas. La estructura básica de una tabla HTML incluye varias etiquetas, como <table> para definir la tabla en sí, <tr> para las filas, <th> para las celdas de encabezado y <td> para las celdas de datos. Ver Ejemplo
13 <div> Se utiliza como un contenedor genérico para agrupar y estructurar otros elementos HTML. La palabra "div" proviene de "division", y la etiqueta se utiliza para dividir o agrupar secciones de contenido en una página web. La etiqueta &t;div> en sí misma no tiene un impacto visual directo, pero es fundamental para organizar y dar estructura al diseño de una página. Ver Ejemplo
14 <header> El elemento <header> no debe ser confundido con <head> usado antes para construir la cabecera del documento. Del mismo modo que <head>, la intención de <header> es proveer información introductoria (títulos, subtítulos, logos), pero difiere con respecto a <head> en su alcance. Mientras que el elemento <head> tiene el propósito de proveer información acerca de todo el documento, <header> es usado solo para el cuerpo o secciones específicas dentro del cuerpo. Ver Ejemplo
15 <nav> Representa una sección de una página cuyo propósito es proporcionar enlaces de navegación, ya sea dentro del documento actual o a otros documentos. Ver Ejemplo
16 <section> Representa una sección genérica independiente de un documento, que no tiene un elemento semántico más específico para representarla. Ver Ejemplo
17 <aside> Representa una sección de una página que consiste en contenido que está indirectamente relacionado con el contenido principal del documento. Ver Ejemplo
18 <footer> Representa un pie de página para el contenido de sección más cercano o el elemento raíz de sección. Ver Ejemplo
19 <article> Utilizada para definir un contenido autónomo y de importancia en una página web. Ver Ejemplo
20 <hgroup> Representa el encabezado de una sección. Define un solo título que participa de la estructura del documento como el encabezado de la sección implícita o explícita a la que pertenece. Ver Ejemplo
21 <figure> Esta etiqueta se utiliza para envolver cualquier contenido multimedia, como imágenes, gráficos, videos, entre otros. Proporciona un contexto semántico al contenido y permite que se pueda referenciar fácilmente. Ver Ejemplo
22 <figcaption> Esta etiqueta se utiliza dentro de un elemento <figure> para proporcionar una descripción o leyenda al contenido multimedia que está envuelto por la etiqueta <figure>. Ver Ejemplo
